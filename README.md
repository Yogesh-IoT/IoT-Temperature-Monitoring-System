# IoT Temperature Monitoring System

## 📌 Description
This project uses Raspberry Pi Pico to read temperature and sends data to ESP32 via UART. ESP32 uploads data to ThingSpeak cloud for real-time monitoring.

## ⚙️ Components
- ESP32 Dev Module
- Raspberry Pi Pico
- Temperature Sensor
- WiFi (Mobile Hotspot)

## 🚀 Features
- Real-time temperature monitoring
- Cloud data logging using ThingSpeak
- Serial communication (UART)

## 📊 Output
Live data visualization using ThingSpeak graphs

## 🔮 Future Scope
- IFTTT alerts
- Buzzer alert system
- OLED display
